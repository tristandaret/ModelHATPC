# LUT

Tools to create and visualize lookup tables (LUTs) used by the HATPC model.

Files include:
- `HAT_dEdx_LUT_maker.py`, `LUT_maker_vTTree.py` - scripts to generate LUTs.
- `LUT_visualizer.py` - quick visualization tools for produced tables.
- `Ratio_estimator.py` - utilities used to estimate charge/energy ratios.

LUT generation may require additional data files or inputs not included in the
repository. Inspect each script header before running.
